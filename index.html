AI Sleep Detector 🚗

A Real-Time Driver Drowsiness Detection System built using Python, OpenCV and MediaPipe.

View Source Code

About the Project

This system detects eye closure using Eye Aspect Ratio (EAR). If the driver's eyes remain closed for more than 1 second, an alarm is triggered to prevent accidents.

Technologies Used

Python • OpenCV • MediaPipe • NumPy • Pygame

Features

Real-time face detection
Eye tracking using facial landmarks
Automatic alarm system
Accident prevention focused
